# North Star Metric

A strategic alignment framework that identifies the one metric most closely correlated with the long-term value a product delivers to its customers. The North Star Metric (NSM) serves as the focal point for the entire company — every team's work should ultimately ladder up to moving this number. It bridges the gap between daily feature work and long-term business outcomes.

## Steps

1. **Articulate the Core Value** — Define, in plain language, the primary value customers get from the product. What moment or outcome makes them say "this is worth it"? This is not a business outcome (revenue) but a customer outcome (problem solved, progress made).

2. **Generate Metric Candidates** — Brainstorm metrics that quantify how much of that core value is being delivered. Good NSM candidates reflect: value delivered to the customer (not extracted from them), breadth (how many users), depth (how much value per user), and frequency (how often). Examples: weekly active users completing a key action, messages sent, bookings completed, queries answered.

3. **Evaluate Against Criteria** — Score each candidate on six criteria: (a) reflects customer value, (b) is measurable and available, (c) is a leading indicator of revenue, (d) the team can influence it, (e) is understandable by everyone in the company, (f) is not easily gamed. Eliminate metrics that fail on any critical criterion.

4. **Select the North Star Metric** — Choose the single metric that best balances all criteria. Resist the urge to pick two. If no single metric works, the core value statement likely needs sharpening. Document why this metric was chosen over alternatives.

5. **Define Input Metrics** — Break the NSM into 3-5 input metrics that different teams can directly influence. These form a tree: the NSM is the trunk, inputs are branches. Example: if NSM = weekly active subscribers, inputs might be new subscriber activation rate, content publish frequency, and churn rate.

6. **Operationalize and Review** — Build dashboards, set targets, and review the NSM and inputs in a regular cadence (weekly or biweekly). Re-evaluate the NSM quarterly — it should evolve as the product and market mature, but not change so often that it loses alignment power.

## Output Format

A structured report containing:

- **Core Value Statement**: One sentence describing the primary value the product delivers to customers
- **Metric Candidates**: List of 5-10 candidates with pros, cons, and evaluation scores
- **Selected North Star Metric**: The chosen metric with definition, measurement method, and rationale
- **Input Metric Tree**: 3-5 input metrics with ownership assignments and relationship to the NSM
- **Current Baseline**: The current value of the NSM and each input metric
- **Target and Timeline**: Where the team aims to move the NSM within a defined time horizon
- **Review Cadence**: How often the metric will be reviewed and under what conditions it should be reconsidered
